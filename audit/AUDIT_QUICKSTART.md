# Audit Quickstart

This guide summarizes the audit process and evidence for strict compliance.

- All code, tests, and docs are in English.
- JavaDoc only for backend code.
- PRs, reviews, merges, and CI/CD are managed exclusively on GitHub.
- Gitea is used only as a mirror (no CI/CD, no PR).
- SonarQube, ESLint, Checkstyle, PMD, SpotBugs: zero warning.
- Bonus features: wishlist, multi-payment.

See README.md for checklist and evidence.
